이 글은 C 열거형(enum)을 공부하고 정리한 글입니다
{: .notice--warning}

```c
enum food { 
    FOOD_MILK,
    FOOD_PIZZA,
    FOOD_CAKE
};

int main(void)
{
    food favorite_food = FOOD_PIZZA;
    
    return 0;
}
```
- enum은 정수에 별명을 붙인 것이다
- int와 섞어서 사용이 가능하다 **(명시적인 캐스팅이 없어도)**
  - `int -> enum` `enum -> int` `enum -> 또 다른 enum` 가능하다
